# Phase 2 — Technical Skills

> Estimated Completion Time: 4–8 Months  
> Difficulty Level: Intermediate  

## Objective
Develop applied security capabilities across network, system, and web application domains through structured theory and lab-driven practice.

---

# 1️⃣ Network Security

## Core Theory
- Network Attack Surface Mapping
- ARP Spoofing, MITM
- DNS Attacks
- Firewall Architectures
- IDS/IPS Concepts
- VPN & Secure Tunneling

## Courses
- Professor Messer Network+  
  https://www.professormesser.com/network-plus/n10-008/n10-008-training-course/
- SANS SEC560 – Network Penetration Testing  
  https://www.sans.org/cyber-security-courses/network-penetration-testing-ethical-hacking/
- Cybrary Network Security  
  https://www.cybrary.it/course/cyber-network-security

## Books
- Network Security Essentials — William Stallings  
- Practical Packet Analysis — Chris Sanders  
- The Practice of Network Security Monitoring — Richard Bejtlich  

## Tools
- Nmap  
  https://nmap.org/
- Wireshark  
  https://www.wireshark.org/
- Zeek  
  https://zeek.org/
- Suricata  
  https://suricata.io/
- pfSense  
  https://www.pfsense.org/

## Practical Labs
- Perform full network reconnaissance with Nmap
- Detect ARP spoofing using Wireshark
- Deploy Suricata IDS in a lab
- Configure pfSense firewall rules
- Simulate MITM attack in controlled environment

## Mastery Benchmark
You should be able to:
- Map network attack surface
- Interpret IDS alerts
- Design firewall segmentation strategy
