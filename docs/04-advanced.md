# Phase 4 — Advanced Security Domains

> Estimated Completion Time: 8–18 Months  
> Difficulty Level: Advanced → Expert  

## Objective
Develop deep analytical capability in malware analysis, threat intelligence, and nation-state level attack research.

---

# 1️⃣ Malware Analysis & Reverse Engineering

## Core Domains
- Static Analysis
- Dynamic Analysis
- Windows Internals
- PE File Format
- Memory Forensics
- Shellcode Analysis
- Obfuscation & Packing Techniques

## Foundational Knowledge Required
- Strong C/C++
- Assembly Basics (x86/x64)
- Operating System Internals

## Courses & Training
- Practical Malware Analysis Labs  
  https://www.nostarch.com/malware
- SANS FOR610 — Reverse Engineering Malware  
  https://www.sans.org/cyber-security-courses/reverse-engineering-malware-malware-analysis-tools-techniques/
- OpenSecurityTraining — Intro to x86  
  http://opensecuritytraining.info/IntroX86.html

## Books
- Practical Malware Analysis — Sikorski & Honig  
- Malware Analyst’s Cookbook  
- Learning Malware Analysis — Monnappa K A  

## Tools
- Ghidra  
  https://ghidra-sre.org/
- IDA Pro  
  https://hex-rays.com/ida-pro/
- x64dbg  
  https://x64dbg.com/
- Cuckoo Sandbox  
  https://cuckoosandbox.org/
- REMnux  
  https://remnux.org/

## Practical Labs
- Analyze packed malware sample
- Reverse engineer simple keylogger
- Perform memory dump analysis
- Build automated sandbox pipeline

## Mastery Benchmark
You should be able to:
- Reverse engineer Windows malware
- Extract IOCs manually
- Analyze obfuscated payloads
- Produce professional malware report
