# Phase 1 — Foundation

> Estimated Completion Time: 3–6 Months  
> Difficulty Level: Foundational → Intermediate  

## Objective
Build deep computing fundamentals required for advanced cybersecurity domains.

---

# Mastery Outcomes

By completing Phase 1, you should be able to:

- Analyze packet captures without relying entirely on GUI tools  
- Write network automation scripts  
- Explain memory management and process scheduling  
- Configure and harden a Linux system  
- Understand TLS handshake and basic cryptographic primitives
